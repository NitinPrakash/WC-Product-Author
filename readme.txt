WC Product Author enables author functionality for Woocommerce Products, Author can be assigned to Woocommerce Product using this plugin

== Description ==

Woocommerce Product Author is an open source plugin that allows admin to add Author to Woocommerce Product.
This plugins is compatible with latest wordpress, woocommerce version, PHP 5.6.x, PHP 7.x and Gutenberg.

== Installation ==

* Upload the directory '/wc-product-author/' to your WP plugins directory and activate from the Dashboard of the main blog.
* All done, nothing needs to be done now.
